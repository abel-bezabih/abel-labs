# Abel Labs Logo Assets

This folder contains downloadable logo files for Abel Labs.

## Files Included

### 1. `abel-labs-logo.svg`
- **Full color version** with animated gradient colors
- Best for: Websites, digital use, presentations
- Colors: Cyan → Blue → Purple → Fuchsia → Red → Orange → Yellow gradient

### 2. `abel-labs-logo-white.svg`
- **White version** for dark backgrounds
- Best for: Dark themes, dark backgrounds, inverted designs
- Colors: White gradient on transparent background

### 3. `abel-labs-logo-simple.svg`
- **Simple solid color version** (no gradients)
- Best for: Print, single-color applications, fax, embroidery
- Colors: Dark gray text with blue icon

## Usage Guidelines

### Digital Use
- Use the full color SVG (`abel-labs-logo.svg`) for websites and digital media
- Use the white version (`abel-labs-logo-white.svg`) on dark backgrounds

### Print Use
- Use the simple version (`abel-labs-logo-simple.svg`) for print materials
- Ensure minimum size of 100px width for readability

### File Formats
- **SVG**: Scalable vector format - use for web and print
- **PNG**: Raster format - export from SVG if needed for specific use cases

## How to Download

1. **Right-click** on any SVG file
2. Select **"Save As"** or **"Download"**
3. Choose your destination folder
4. The file will be saved to your computer

## Converting to PNG

If you need PNG versions:

1. Open the SVG file in a browser (Chrome, Firefox, Safari)
2. Right-click → "Inspect" or "Inspect Element"
3. Take a screenshot, or
4. Use an online converter like:
   - https://convertio.co/svg-png/
   - https://cloudconvert.com/svg-to-png
   - Or use design software like Figma, Adobe Illustrator, or Inkscape

## Logo Specifications

- **Icon**: Code symbol (angle brackets)
- **Text**: "Abel" in bold
- **Tag**: "Labs" in a light-colored bar
- **Font**: Arial (or similar sans-serif)
- **Aspect Ratio**: ~3.3:1 (width:height)
